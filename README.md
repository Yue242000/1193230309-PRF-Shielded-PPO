# 1193230309-PRF-Shielded-PPO
PRF-Shielded PPO 移动机器人动态路径规划——实验视频
本仓库用于展示课程大作业《基于预测风险人工势场与安全动作屏蔽的PPO移动机器人动态路径规划研究》的仿真实验视频。
本文提出的 PRF-Shielded PPO 由三个核心部分组成：
预测风险人工势场（PRF）：根据动态障碍物的相对位置和速度计算 TTC/DCPA 风险，提前识别可能发生的近距离交互；
残差式 PPO 策略：在 PRF 基础动作上学习速度与转向残差，用于修正人工势场并处理复杂动态场景；
安全动作屏蔽（Safety Shield）：在动作执行前预测短期轨迹，必要时从候选动作中选择与原动作偏差较小的安全动作。
视频主要展示机器人路径、动态障碍运动、局部感知、安全距离、动作修正以及时间、路径、能耗等运行指标。

推荐观看顺序
如果只观看一段视频，建议首先观看 视频 01，它展示了本文完整创新方法在快速动态障碍场景中的运行过程。随后可观看视频 06 和视频 07，了解不同算法以及 Safety Shield 开启与关闭时的行为差异。

画面信息说明
机器人及运动轨迹：表示机器人已经执行的实际路径；
静态障碍和动态障碍：动态障碍的位置随仿真时间同步更新；
目标点：机器人需要到达的终点；
预测轨迹/风险信息：用于显示短期运动趋势与动态接近风险；
Shield 状态：表示当前动作是否被安全层修正；
运行指标：包括到达时间、路径长度、能耗代理、最小安全间距等。
其中，能耗为根据线速度和角速度计算的运动学代理指标，并不等同于真实机器人电机功耗。

实验条件
二维连续动态导航环境；
差速移动机器人，控制周期为 0.1 s；
局部观测包含目标相对信息、24 束 LiDAR 距离及距离变化率等；
视频使用完成约 500k 环境步训练后的正式模型，以确定性策略生成；
单场景视频按照真实仿真时间播放，对比视频按照归一化回合进度同步展示。
