# PRF-Shielded PPO 移动机器人动态路径规划实验视频
本仓库用于展示课程大作业《基于预测风险人工势场与安全动作屏蔽的 PPO 移动机器人动态多指标路径规划研究》的仿真实验视频。

## 项目简介
本文提出的 **PRF-Shielded PPO** 包含三个核心模块：
1. **预测风险人工势场（PRF）**：通过动态障碍的相对位置和速度计算 TTC/DCPA 风险，实现提前避障。
2. **残差式 PPO 策略**：在 PRF 基础动作上学习速度和转向残差，修正人工势场并处理复杂场景。
3. **安全动作屏蔽（Safety Shield）**：在动作执行前预测短期轨迹，并在必要时选择偏差较小的安全候选动作。
视频主要展示机器人路径、动态障碍运动、局部感知、安全距离、Shield 动作修正以及时间、路径和能耗等指标。

## 画面信息说明
- **机器人及其轨迹**：表示机器人已经执行的实际路径。
- **静态障碍与动态障碍**：动态障碍位置随仿真时间同步更新。
- **目标点**：机器人需要到达的终点。
- **预测轨迹与风险信息**：表示机器人和动态障碍的短期运动趋势。
- **Shield状态**：显示当前控制动作是否被安全层修正。
- **运行指标**：包括路径长度、到达时间、能耗代理和最小安全间距等。
其中，能耗是根据线速度和角速度计算的运动学代理指标，不等同于真实机器人电机功耗。

## 实验条件
- 二维连续动态导航环境。
- 差速移动机器人，控制周期为0.1 s。
- 局部观测包含目标相对信息、24束LiDAR距离及距离变化率等。
- 视频采用完成约500k环境步训练后的正式模型生成。
- 单场景视频按照实际仿真时间播放。
- 多算法对比视频按照归一化回合进度同步展示。
